Mi Web

Vamos a hacer esta cosa, basicamente vamos a ver para que sirven las etiquetas princpiales en html.

Empezando por la etiqueta <!DOCTYPE html> que define el tipo de documento y la versión de HTML.

Esta tambien la etiqueta de <html> que basicamente es el contenedor de toda nuestra página web.

Finalmente estan las etiquetas de cabecera, definidas como <head>, que contiene elementos como el titulo de nuestra página.

Encabezados

Los encabezados en HTML van desde <h1> (más importante) hasta <h6> (menos importante).

Ejemplo de encabezados:

<h1> Encabezado 1 </h1>

<h2> Encabezado 2 </h2>

<h3> Encabezado 3 </h3>

<h4> Encabezado 4 </h4>

<h5> Encabezado 5 </h5>
<h6> Encabezado 6 </h6>

Listas

Atributos Principales

  • src: Fuente /(ruta de una imagen o web). Ejemplo: <img src="foto.jpg"> <img src="https://mi.imagen.com">
  • alt: Texto alternativo de una imagen. Ejemplo: <img alt="Descripción">
  • href: Destino de un enlace. Ejemplo: <a href="https://www.utez.edu.mx:8443/SISAVA/">
  • title: Título emergente al pasar el mouse. Ejemplo: <a title="Más info">Enlace</a>
  • class y id: Para identificar o aplicar estilos CSS.
  • style: Permite aplicar estilos CSS directamente a un elemento.
